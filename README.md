# League of Legends 승률 예측 

리그오브레전드 랭크게임 승률 예측 AI 적용 웹페이지


### 웹페이지 소개

- 웹페이지에 특정 판의 정보를 입력하면, 해당 게임의 승리할 확률을 알려줍니다.

- 이를 통해, 본래 질 게임이었는데 이겼거나, 이겨야 할 게임인데 질 경우를 확인할 수 있습니다.

---

### 웹페이지 실행 방법(로컬)

Local에서 실행하려 할 경우, 다음 명령어를 입력해주시기 바랍니다.

```bash
  python main.py
```
